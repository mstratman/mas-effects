# Site setup

## Source files:

* `nunjucks/` - [nunjucks](https://mozilla.github.io/nunjucks/) files, written as html to `docs/`
* `scss/` - written as css to `docs/css`
* `static/` - copied to `docs/`. e.g. `docs-src/static/imgs/*` ends up in `docs/imgs/`
* `js/`

## Dev: 

* `npm run dev`

## Prod

* `npm run build`
